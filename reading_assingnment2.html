Cheatsheet 2

HTML

  1. Name the components of html elements properly: Doctype, Comments, Elements or Tags, Attributes, Frames
  2. Identify void elements and how they are denoted: paragraph, has no attribute or co.
  3. Identify attributes: An HTML Attribute sets either style or layout or content related information for an HTML Element.
  4. Write down the correct DOCTYPE declaration for html 5: !DOCTYPE html --dont forget the smaller and bigger
  5. Write down syntactically correct statements for the following elements: Article tag specifies independent, self-contained content. body contains all the contents of an HTML document. line break inserts a single line break. headings define HTML headings. Top level element (html) tells browser that html doc. root of html doc. container for all html elements. Image named jpg paragraph defines a paragraph section defines sections in a document, such as chapters, headers, footers label appearing in middle bar(title) search-engine results, link to pages(a) defines a hyperlink comment used to insert comments in the source code head with title and meta elements container for all the head elements.

CSS

  1. Name the components of a css rule properly: selector, property, value
  2. Use combinators properly: direct child (article > p , if straight after article is an p it will be like css says) descendant (article h2, are very similar to child selectors, except that child selectors only select direct descendents), adjacent sibling(h2 + p, only together, so standing under each other, css works), sibling (h2 ~ p, p elements that follows and share same parent)
  3. Specify colors by color names and in hexadecimal notation: "red" or "#ffffff" or "rgb(255,255,255)"
  4. Declarations for color, background color: "color: red;" "background-color:blue;"
  5. Pseudo classes for the element: special state of an element a:link, a:visited, a:hover, a:active
  6. Declaration for background image: background-image: url("paper.gif");
  7. Declaration for text alignment (left, right, center, justify): text-align: center;
  8. Declaration for text decoration (overline, underline, line-through): text-decoration: overline;
  9. Declaration for text transformation (uppercase, lowercase, capitalize): text-transform: uppercase;
  10. Identify the difference between serif and sans-serif fonts: serif is the cool old looking thing, really decorative. sansserif is computer.
  11. Declarations for font families: font-family: "Times New Roman", Times, serif;
  12. Declarations for font style normal and italic: font-style: normal;
  13. Declarations for font sizes: font-size: 40px;
  14. Declarations for font weights: font-weight: normal;
